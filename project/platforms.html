Two lines. Every wearable.

One OAuth flow your user sees once. Normalised, time-aligned observations stream into your app. Drop the dozen vendor SDKs.

// 1. Install
$ npm install @biotrackos/sdk

// 2. Connect every wearable through one flow
import { BiotrackOS } from '@biotrackos/sdk';

const bt = new BiotrackOS({ apiKey: process.env.BT_KEY });

await bt.connect(userId, {
  sources: ['apple', 'oura', 'whoop', 'garmin'],
  redirect: 'https://your.app/connected'
});

Subscribe to the stream.

One canonical schema. Resting HR is resting HR — whether it came from Apple, Garmin, or Polar. Dedup, gaps, and conflicts are already resolved.

bt.observations.subscribe({
  userId,
  metrics: ['hrv', 'sleep_efficiency', 'rhr'],
  since: 'PT24H'
}, (obs) => {
  console.log(obs);
  // {
  //   metric: 'hrv',
  //   value: 62,
  //   unit: 'ms',
  //   source: 'oura',
  //   ts: '2026-05-11T08:42:11Z',
  //   confidence: 0.94
  // }
});
